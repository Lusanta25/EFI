## 驱动情况

- 显卡驱动HD4600

  ~~显卡驱动能正常运作，但是每次开机都会`花屏`，只有按一下`电源键`让它睡眠，然后再唤醒才能不花屏。~~

  花屏问题已经解决：**只需要在bios设置里，将启动模式 boot mode设置为传统模式启动，并选择uefi first就可以不花屏启动。**
